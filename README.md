# eve-echos
Eve Echos neo4j

This is an attempt to create a map of routes between ITCs in the eve echos mobile game
http://www.eveechos.com

This was created in GraphCommons and exported to a variety of formats:
json
neo4j cypher
CSV
graphml

The original is here:
